# モバイルアプリのプロトタイプ

Next.jsとTailwind CSSを使用したモバイルアプリのプロトタイプです。レスポンシブデザインでモバイルとデスクトップの両方をサポートしています。

## 機能

- モバイル/デスクトップ表示の自動切り替え
- ダークモード対応
- 3つの画面（ホーム、プロフィール、設定）

## 実行方法

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```
